## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/en/) (v18.0.0 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Harshb234/no-cap-dict.git
   ```

2. **Navigate into the directory**
   ```bash
   cd no-cap-dict
   ```

3. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

<br />

## 💻 Development

To start the local development server with Hot Module Replacement (HMR):

```bash
npm run dev
# or
yarn dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.

<br />

## 📦 Production Build

To bundle the application for deployment:

```bash
npm run build
# or
yarn build
```
This will generate an optimized, production-ready build in the `dist/` directory.
